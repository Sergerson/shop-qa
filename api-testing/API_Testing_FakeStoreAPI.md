API тест - FakeStoreAPI - тестовый магазин

В рамках проекта выполнено ручное API-тестирование учебного сервиса FakeStoreAPI с использованием Postman. 

Цель проекта - показать базовые навыки работы с API: отправка запросов, проверка статус-кодов, анализ JSON-ответов
