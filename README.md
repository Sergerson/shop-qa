**Разрешение экрана:** 1920x1080 (Full HD)

## Чек-листы модулей
## Чек-лист "Авторизация и регистрация"

### Модуль: Авторизация и Регистрация 

#### Поле ввода: Email

#### Позитивные проверки 
- [ ] Успешная регистрация нового пользователя с валидными данными (test@mail.com, pass: testpass123)
- [ ] Успешная авторизация (вход) зарегистрированного пользователя
- [ ] Код верификации приходит на email пользователя
- [ ] Нажать кнопку "выйти из системы" в Личном кабинете -> разлогин пользователя, перенос на главную магазина
- [ ] Ввести корректный тестовый email (test@mail.com), но с пробелом (test@mail. com) -> успех (работает trim)
- [ ] Кнопка "Отмена" в окне "Регистрация" закрывает данное окно

#### Негативные проверки 
- [ ] Ввести неверный код верификации в после подтвеждения регистрации на сайте ИМ -> ошибка "Введите верный код верификации"
- [ ] Оставить пустым поле верификации и нажать кнопку подтвердить -> ошибка "Заполните поле"
- [ ] При регистрации в поле email вводится только буквы без @ -> ошибка "Введите корректный email" для входа
- [ ] При регистрации в поле email вводится только цифры -> ошибка "Введите корректный email" для входа
- [ ] При регистрации оставить поле Email пустым -> ошибка "Пожалуйста, заполните поля"
- [ ] При регистрации использовать одинаковый email и пароль (test@mail.com и test@mail.com)
- [ ] Ввести только ".com" -> ошибка "Введите корректный email"
- [ ] Ввести специальные символы !@#^% -> ошибка "Введите корректный email"
- [ ] Попытка зарегистрироваться еще раз с почтой, которая уже зарегистрирована -> ошибка "Такой пользователь уже существует"

#### Поле ввода: Пароль (по ТЗ пароль должен быть длиной от 6 до 15 символов, латинские буквы и цифры)

##### Позитивные проверки 
- [ ] Можно скрыть/показать пароль (иконка глаза)
- [ ] Ввести латинские буквы и цифры (внутри валидного диапазона) -> успех
- [ ] Ввести ровно 6 символов (буквы и цифры) -> успех
- [ ] Ввести ровно 15 символов (буквы и цифры) -> успех

##### Негативные проверки 
- [ ] Поле пустое -> ошибка "Заполните все поля"
- [ ] Только латинские буквы -> ошибка "Пароль должен состоять из латинских букв и цифр"
- [ ] Ввести специальные символы !@#^% -> ошибка "Пароль должен состоять из латинских букв и цифр"
- [ ] Ввести только цифры -> ошибка "Пароль должен состоять из латинских букв и цифр"
- [ ] Ввести латинские буквы и специальные символы -> ошибка "Пароль должен состоять из латинских букв и цифр"
- [ ] Ввести 5 символов (буквы и цифры) -> ошибка "Пароль должен быть длиной от 6 до 15 символов"
- [ ] Ввести 16 символов (буквы и цифры) -> ошибка "Пароль должен быть длиной от 6 до 15 символов"
- [ ] Ввод SQL-инъекции (например, `' OR '1'='1`) -> ошибка валидации
- [ ] Ввод XSS-скрипта (<script>alert(1)</script>) -> скрипт не выполняется -> ошибка
